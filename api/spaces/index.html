Spaces

Hilbert spaces form the mathematical foundation of quantum mechanics. QSymbolic.jl provides types to represent both simple and composite spaces.

Overview

TypeDescription
HilbertSpaceA single named Hilbert space (finite or infinite-dimensional)
CompositeSpaceTensor product of two or more spaces
FockSpaceConvenience constructor for infinite-dimensional spaces

Types

QSymbolic.AbstractSpace — Type
AbstractSpace{name, dim}

Abstract supertype for all quantum state spaces.

Type parameters:

  • name: Tuple of symbols identifying the space
  • dim: Tuple of dimensions (or nothing for infinite-dim)
source
QSymbolic.HilbertSpace — Type
HilbertSpace(name::Symbol, dim=nothing)

A named Hilbert space with optional finite dimension.

Examples

julia> H = HilbertSpace(:H, 2)  # qubit
ℋ(H, dim=2)

julia> F = HilbertSpace(:F)     # infinite-dim
ℋ(F)
source
QSymbolic.CompositeSpace — Type
CompositeSpace(space1, space2)

Tensor product of two spaces. Usually created via space1 ⊗ space2.

Examples

julia> H1 = HilbertSpace(:A, 2);

julia> H2 = HilbertSpace(:B, 3);

julia> H1 ⊗ H2
ℋ(A) ⊗ ℋ(B)
source
QSymbolic.FockSpace — Function
FockSpace(name::Symbol)

Create an infinite-dimensional Fock space (number state basis). Equivalent to HilbertSpace(name, nothing).

Examples

julia> F = FockSpace(:F)
ℋ(F)
source

Tensor Product

Use ⊗ (typed as \otimes<tab>) to create composite spaces:

QSymbolic.:⊗ — Method
⊗(space1::AbstractSpace, space2::AbstractSpace)

Tensor product of two Hilbert spaces.

source

Examples

using QSymbolic

# Single qubit
H = HilbertSpace(:qubit, 2)
Hb = Basis(H, :default)

# Harmonic oscillator (infinite-dimensional)  
F = FockSpace(:oscillator)

# Two-qubit system
H_AB = HilbertSpace(:A, 2) ⊗ HilbertSpace(:B, 2)
