Symbolic Scalars

The symbolic scalar system provides lazy arithmetic evaluation, enabling truly symbolic quantum computations.

Overview

AbstractSymbolic

Symbolic Variable

QSymbolic.Sym — Type
Sym(name::Symbol)

A symbolic variable. Arithmetic operations build expression trees.

Examples

n = Sym(:n)
expr = √n * 2        # √n · 2
substitute(expr, :n => 4)  # → 4.0

Expression Tree

QSymbolic.SymExpr — Type

A symbolic expression representing op(args...). Operations are stored as symbols: :+, :-, :*, :/, :^, :sqrt, :conj, etc.

Evaluation Functions

Substitution

QSymbolic.substitute — Function
substitute(expr, pairs...)
substitute(expr, dict)

Substitute symbolic variables with values.

Examples

n = Sym(:n)
expr = √n + 1
substitute(expr, :n => 4)  # → √4 + 1 (still symbolic with SymNum)
source

Numeric Evaluation

QSymbolic.evaluate — Function
evaluate(expr)

Evaluate a symbolic expression to a numeric value. All symbols must have been substituted.

Examples

n = Sym(:n)
expr = √n + 1
result = substitute(expr, :n => 4)
evaluate(result)  # → 3.0

Supported Operations

The following operations are defined for AbstractSymbolic types and build expression trees:

Arithmetic

  • +, - (binary and unary)
  • *, /, //
  • ^

Functions

  • sqrt, √
  • conj, adjoint (')
  • abs, abs2
  • sin, cos, tan
  • exp, log

Examples

Basic Usage

using QSymbolic

# Create symbolic variables
n = Sym(:n)
θ = Sym(:θ)

# Build expressions
expr = √n + 1
trig = sin(θ)^2 + cos(θ)^2

# Substitute and evaluate
result = substitute(expr, :n => 4)
evaluate(result)  # → 3.0

Introspection

a, b = Sym(:a), Sym(:b)
expr = a^2 + 2*a*b + b^2

symbols(expr)        # → Set([:a, :b])
is_numeric(expr)     # → false
is_numeric(substitute(expr, :a => 1, :b => 2))  # → true

Simplification

n = Sym(:n)

simplify(n * 1)  # → n
simplify(n * 0)  # → 0
simplify(n + 0)  # → n
simplify(n^0)    # → 1
