States

Quantum states are represented as kets |ψ⟩ and their duals as bras ⟨ψ|. QSymbolic.jl provides types for basis states, superpositions, and product states.

Overview

TypeDescription
Ket / BraSingle basis state
WeightedKet / WeightedBraScalar × basis state
SumKet / SumBraLinear combination of basis states
ProductKet / ProductBraTensor product of states (order-independent)

Index Types

Kets support flexible indices:

# Single index
Ket(basis, :ψ)           # Symbol
Ket(basis, 0)            # Integer (Fock states)
Ket(basis, Sym(:n))      # Symbolic

# Multi-index (for composite bases)
Ket(basis, (n, m))       # Tuple of indices
Ket(basis, (n, m, k))    # Arbitrary length

Basis States

QSymbolic.Ket — Type
Ket{B<:AbstractBasis}(index)
Ket(basis, index)

Basic ket state |index⟩ in a specific basis. This is the fundamental building block.

Note: A basis is required. You must explicitly provide a basis object.

Index can be:

  • A Symbol (e.g., :↑, :ψ)
  • An Integer (e.g., 0, 1, 2 for Fock states)
  • A symbolic expression (Sym, SymExpr)
  • A tuple for multi-index (e.g., (:n, :m) or (0, 1) for composite bases)

Examples

julia> H = HilbertSpace(:H, 2);

julia> Zb = Basis(H, :z);

julia> up = Ket(Zb, :↑)   # |↑⟩ in z-basis
|↑⟩

julia> Xb = Basis(H, :x);

julia> ψ = Ket(Xb, :ψ)     # |ψ⟩ in x-basis
|ψ⟩

See also: Bra, Basis, WeightedKet, SumKet, ProductKet

source
QSymbolic.Bra — Type
Bra{B<:AbstractBasis}(index)
Bra(basis, index)

Basic bra state ⟨index| in a specific basis. Usually created via adjoint: ket'.

Note: A basis is required. You must explicitly provide a basis object.

See also: Ket, Basis

Weighted States

QSymbolic.WeightedKet — Type
WeightedKet{B<:AbstractBasis, T}(ket, weight)

A ket (Ket or ProductKet) multiplied by a scalar weight. Created automatically via scalar multiplication: weight * ket.

Examples

julia> _, Hb = HilbertSpace(:H, 2);

julia> ψ = Ket(Hb, :ψ);

julia> 2 * ψ
2·|ψ⟩

See also: Ket, SumKet

source
QSymbolic.WeightedBra — Type
WeightedBra{B,T}(bra, weight)

A bra multiplied by a scalar weight. Created via adjoint of WeightedKet (with complex conjugate weight).

See also: WeightedKet

source

Superpositions

QSymbolic.SumKet — Type
SumKet{B<:AbstractBasis, T}(kets, weights; name=nothing)

Linear combination of kets with weights: Σᵢ wᵢ|ψᵢ⟩. Created automatically via addition/subtraction of kets.

The kets can be either Ket or ProductKet (but all must share the same basis structure).

Examples

julia> _, Hb = HilbertSpace(:H, 2);

julia> ψ, ϕ = Ket(Hb, :ψ), Ket(Hb, :ϕ);

julia> ψ + ϕ
|ψ⟩ + |ϕ⟩

julia> (ψ + ϕ) / √2
0.7071067811865475·|ψ⟩ + 0.7071067811865475·|ϕ⟩

See also: Ket, WeightedKet

source
QSymbolic.SumBra — Type
SumBra{B,T}(bras, weights; name=nothing)

Linear combination of bras with weights. Created via adjoint of SumKet (with complex conjugate weights).

See also: SumKet

source

Composite States

For multi-particle systems. ProductKet is order-independent (bosonic).

QSymbolic.ProductKet — Type
ProductKet{Bs<:Tuple}(kets::Vector{Ket})
ProductKet(ket1, ket2, ...)

Tensor product of kets: |ψ₁⟩⊗|ψ₂⟩⊗...⊗|ψₙ⟩. Lives in a CompositeBasis. This is a container struct (like SumKet), not a basic element - only Ket is a basic element.

Note: ProductKets are order-independent (bosonic/symmetric). The kets are canonically ordered by basis, so k1 ⊗ k2 == k2 ⊗ k1 when they have different bases. This is the default behavior; fermionic (anti-symmetric) tensor products will be added in a future update.

The tensor product operator ⊗ creates ProductKets automatically.

Examples

julia> _, B1 = HilbertSpace(:A, 2); _, B2 = HilbertSpace(:B, 2);

julia> ψ, ϕ = Ket(B1, :ψ), Ket(B2, :ϕ);

julia> ψ ⊗ ϕ
|ψ⟩⊗|ϕ⟩

julia> ϕ ⊗ ψ == ψ ⊗ ϕ  # Order-independent (bosonic)
true

julia> _, B3 = HilbertSpace(:C, 2);

julia> χ = Ket(B3, :χ);

julia> ψ ⊗ ϕ ⊗ χ  # Three kets
|ψ⟩⊗|ϕ⟩⊗|χ⟩

See also: Ket, ⊗

source
QSymbolic.ProductBra — Type
ProductBra{Bs<:Tuple}(bras::Vector{Bra})
ProductBra(bra1, bra2, ...)

Tensor product of bras: ⟨ψ₁|⊗⟨ψ₂|⊗...⊗⟨ψₙ|. Created via adjoint of ProductKet.

Note: ProductBras are order-independent (bosonic/symmetric), matching the behavior of ProductKet.

See also: ProductKet, Bra

Examples

using QSymbolic

H, Hb = HilbertSpace(:H, 2)
Zb = Basis(H, :z)

# Create basis kets
up = Ket(Zb, :↑)
down = Ket(Zb, :↓)

# Superposition
plus = (up + down) / √2

# Adjoint gives bra
up'  # → ⟨↑|

# Inner products
up' * up    # → 1
up' * down  # → 0

# Tensor products (order-independent)
H_A, Ba = HilbertSpace(:A, 2)
H_B, Bb = HilbertSpace(:B, 2)
ψ = Ket(Ba, :ψ)
ϕ = Ket(Bb, :ϕ)
ψ ⊗ ϕ == ϕ ⊗ ψ  # → true

# Symbolic indices
n = Sym(:n, :nonnegative, :integer)
F, Fb = FockSpace(:mode)
ket_n = Ket(Fb, n)  # |n⟩
